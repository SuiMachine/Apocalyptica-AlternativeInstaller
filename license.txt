2)  INSTALLATION
---------------------------------------------------------------------------
Insert the Apocalyptica(tm) CD 1 into your CD-ROM or DVD-ROM drive. After a few seconds, the Autorun Menu will appear. Click on Install to run the installer and follow the on-screen prompts.

In the event the Autorun Menu does not appear after inserting the CD: 
Double-click on the My Computer icon, then double-click on the CD/DVD-ROM icon, and lastly double-click on the Autorun.exe file to bring up the 
Autorun Menu.

You will initially be prompted to select the install path and directory.  The 
default install path is C:\Program Files\Konami\Apocalyptica. The installation
program will give the user the option to install DirectX 8.1. Only install this 
version if you have got a lower version of DirectX running on your machine.

Please note that the game will not install if you attempt to install to a directory that is 256 characters or longer. 

You should now be able to play Apocalyptica(tm). 

Note:If you are using Windows XP or Windows 2000, you must have 
administrator rights to install and run Apocalyptica(tm). 
  

3)  AUTORUN MENU
---------------------------------------------------------------------------
The Autorun Menu will appear any time Apocalyptica(tm) CD 1 is inserted into the 
CD/DVD ROM drive. The menu has several options:

*	Play - only available after you have installed the game, click to start 
	playing Apocalyptica(tm). 
*	Install - only available if you don't have Apocalyptica(tm) installed on your 
	system. Click to install the game. 
*	Uninstall - only available once Apocalyptica(tm) has been installed on your 
	system. Click to uninstall the game.
*	Readme - Instruction guide. 


4)  STARTING THE GAME
---------------------------------------------------------------------------
To begin playing Apocalyptica(tm) insert CD 1 into the CD-ROM or DVD-ROM drive and select Play. Alternatively, click on the Apocalyptica(tm) icon found within the Konami\Apocalyptica program folder. (Start Menu - Programs - Konami -  Apocalyptica). 
 

5)  UNINSTALLING THE GAME
---------------------------------------------------------------------------
If you need to uninstall Apocalyptica(tm) you may do any of the following: 

*	Insert the Apocalyptica(tm) CD 1 and select Uninstall from the Autorun Menu. 
*	Click on the START button, choose Programs, Konami, Apocalyptica(tm) and 
	click on Uninstall Apocalyptica(tm). 
*	Click on the START button, go to Control Panel, Add/Remove 
	Programs and choose Apocalyptica(tm) to uninstall the game.
 

6)  MEMORY AND PERFORMANCE
---------------------------------------------------------------------------
Apocalyptica(tm) requires at least 128 MB of RAM and a Pentium(r) III 800 MHz 
processor; the performance of the game will improve if you have more RAM, a 
faster processor and graphics card. Regardless of your machine specs, you 
can maximize your available memory and increase the overall performance of 
the game by doing the following: 

*	Shut down all non-essential Windows programs especially online chat 
	programs, virus checkers and screen layout utilities. 
*	Regularly defragment your hard disk. 
*	Increase your amount of free hard disk space


7)  APOCALYPTICA(tm) CD SECURITY
---------------------------------------------------------------------------

Apocalyptica(tm) needs to read data from CD 1 from time to time. Please ensure that 
CD 1 is always in your CD drive. Removing the CD may cause Security features to 
be activated which will affect the performance of Apocalyptica(tm)


8)  GRAPHICS 
---------------------------------------------------------------------------

Apocalyptica(tm) requires a graphics card with 3D hardware acceleration and at 
least 32 MB of RAM. The drivers for your graphic card must support DirectX 8.1 
or higher.  If you are having any problems running the game, please ensure 
you have updated your graphics card drivers before contacting technical support.

If  you turn Power Management on in Windows ME systems, the game will display graphical corruption and eventually crash.

Users can only change the colour quality from the Display properties found under
Settings - Control Panel - Display - Display Properties.

If the game is too dark, turn off "Dynamic Lighting" in the options menu under the Video tab.

NOTE:  3Dfx cards  - 3Dfx cards are not officially supported.  Unfortunately, 3Dfx no longer releases updated drivers for their chipsets and do not support certain features of DirectX 8.    
You may be able to find an updated driver at the following web site: http://www.voodoofiles.com/.
 
PLEASE NOTE:  If you choose to install new 3Dfx drivers, you do so at your own risk!


9) MULTIPLAYER AND KEYS 
---------------------------------------------------------------------------

MULTIPLAYER

Users will not be able to play multiplayer games when running behind a firewall or router.

When playing a multiplayer game, all text displayed will appear in the language of the host machine.

KEYS

If a user binds the windows key to any control, every time the user presses that key, the game will perform the function desired as well as change to Windows and launch the Start Menu.

All passwords are case sensitive.

2)  INSTALLATION
---------------------------------------------------------------------------
Legen Sie die Apocalyptica(tm)-CD 1 in Ihr CD-ROM- oder DVD-ROM-Laufwerk ein. Nach einigen Sekunden erscheint das Autorun-Menü. Klicken Sie auf Install (Installieren), um das Installationsprogramm zu starten, und folgen Sie den Anweisungen auf dem Bildschirm.

Falls das Autorun-Menü nach dem Einlegen der CD nicht erscheint: 
Doppelklicken Sie auf das Arbeitsplatz-Icon, dann auf das CD/DVD-ROM-Icon und schließlich auf die Datei Autorun.exe, um das Autorun-Menü aufzurufen.

Zunächst werden Sie aufgefordert, den Pfad und das Verzeichnis für die Installation zu wählen. Der Standardpfad für die Installation ist C:\Programme\Konami\Apocalyptica. Das Installationsprogramm bietet dem Anwender die Möglichkeit, DirectX 8.1 zu installieren. Installieren Sie diese Version nur, wenn auf Ihrem Computer eine ältere Version von DirectX läuft.

Bitte beachten Sie, dass das Spiel nicht installiert wird, wenn Sie versuchen, es in ein Verzeichnis von 256 Zeichen oder länger zu installieren. 

Jetzt können Sie Apocalyptica(tm) spielen. 

Hinweis: Wenn Sie Windows XP oder Windows 2000 verwenden, müssen Sie Administrator-Rechte besitzen, um Apocalyptica(tm) zu installieren und zu starten. 
  

3)  AUTORUN-MENÜ
---------------------------------------------------------------------------
Das Autorun-Menü erscheint, sobald die Apocalyptica(tm)-CD 1 in das CD/DVD-ROM-Laufwerk eingelegt wird. Das Menü hat verschiedene Optionen:

*	Play (Spielen) - erst dann verfügbar, wenn Sie das Spiel installiert haben. Bitte klicken Sie hier, um Apocalyptica(tm) zu starten und zu spielen. 
*	Install (Installieren) - nur verfügbar, solange Sie Apocalyptica(tm) noch nicht auf Ihrem System installiert haben. Bitte klicken Sie hier, um das Spiel zu installieren. 
*	Uninstall (Deinstallieren) - erst dann verfügbar, wenn Sie Apocalyptica(tm) auf Ihrem System installiert haben. Bitte klicken Sie hier, um das Spiel zu deinstallieren.
*	Readme (Liesmich) - Aktuelle Informationen zum Spiel. 


4)  DAS SPIEL STARTEN
---------------------------------------------------------------------------
Um Apocalyptica(tm) zu starten, legen Sie die CD 1 in Ihr CD-ROM- oder DVD-ROM-Laufwerk ein und wählen Sie Play (Spielen). Alternativ können Sie das Apocalyptica(tm)-Icon anklicken, das Sie im Programmverzeichnis Konami\Apocalyptica finden. (Start-Menü - Programme - Konami - Apocalyptica). 
 

5)  DAS SPIEL DEINSTALLIEREN
---------------------------------------------------------------------------
Wenn Sie Apocalyptica(tm) deinstallieren müssen, können Sie auf eine der folgenden Weisen vorgehen: 

*	Legen Sie die Apocalyptica(tm)-CD 1 ein und wählen Sie im Autorun-Menü Uninstall (Deinstallieren). 
*	Klicken Sie die Schaltfläche START an, wählen Sie Programme, Konami, Apocalyptica(tm), und klicken Sie auf Uninstall Apocalyptica(tm) (Apocalyptica(tm) deinstallieren). 
*	Klicken Sie die Schaltfläche START an, gehen Sie auf Systemsteuerung, Software Installieren/Deinstallieren, und wählen Sie Apocalyptica(tm), um das Spiel zu deinstallieren.
 

6)  SPEICHER UND LEISTUNG
---------------------------------------------------------------------------
Apocalyptica(tm) benötigt mindestens 128 MB RAM und einen Pentium(r) III 800 MHz-Prozessor; durch mehr RAM, einen schnelleren Prozessor und eine schnellere Grafikkarte wird die Leistung des Spiels verbessert. Ungeachtet der Spezifikation Ihres Gerätes können Sie Ihren verfügbaren Speicher jedoch folgendermaßen maximieren und die Gesamtleistung des Spiels optimieren: 

*	Schließen Sie alle nicht erforderlichen Windows-Programme, insbesondere Online-Chatprogramme, Virenprüfprogramme und Hilfsprogramme für den Bildschirmaufbau. 
*	Defragmentieren Sie regelmäßig Ihre Festplatte. 
*	Erhöhen Sie Ihren freien Festplattenspeicherplatz. 


7)  APOCALYPTICA(tm) CD-SICHERHEIT
---------------------------------------------------------------------------

Apocalyptica(tm) muss von Zeit zu Zeit Daten von der CD 1 einlesen. Bitte achten Sie darauf, dass sich die CD 1 immer in Ihrem CD-Laufwerk befindet. Das Entfernen der CD kann unter Umständen Sicherheitsmerkmale aktivieren, die die Leistung von Apocalyptica(tm) beeinträchtigen. 


8)  GRAFIK 
---------------------------------------------------------------------------

Für Apocalyptica(tm) sind eine Grafikkarte mit 3D-Hardwarebeschleuniger und mindestens 32 MB RAM erforderlich. Die Treiber für Ihre Grafikkarte müssen DirectX 8.1 oder höher unterstützen. Sollten Sie Probleme haben, das Programm abzuspielen, vergewissern Sie sich bitte, dass Sie Ihre Grafikkartentreiber aktualisiert haben, bevor Sie sich an die technische Unterstützung wenden.

Das Einschalten des Power Managements auf Windows ME-Systemen führt zu grafischen Störungen im Spiel und schließlich zum Absturz des Programms.

Anwender können nur die Farbqualität unter Eigenschaften von Anzeige über Einstellungen - Systemsteuerung - Anzeige - Einstellungen ändern.

Wenn das Spiel zu dunkel ist, deaktivieren Sie die "Dynamische Beleuchtung" im Optionen-Menü unter der Video-Registerkarte.

HINWEIS: 3Dfx-Karten - 3Dfx-Karten werden nicht offiziell unterstützt. Leider gibt 3Dfx keine aktualisierten Treiber für seine Chipsätze mehr aus und unterstützt bestimmte Eigenschaften von DirectX 8 nicht. 
Unter Umständen können Sie unter der folgenden Webseite einen aktualisierten Treiber finden: http://www.voodoofiles.com/.
 
BITTE BEACHTEN SIE: Wenn Sie sich dafür entscheiden, neue 3Dfx-Treiber zu installieren, so geschieht das auf Ihr eigenes Risiko!


9) MULTIPLAYER UND TASTEN
---------------------------------------------------------------------------

MULTIPLAYER

Das Spielen von Multiplayer-Spielen ist nicht möglich, wenn ein Router benutzt wird oder eine Firewall in Betrieb ist.

Beim Spielen eines Multiplayer-Spiels wird der gesamte Text in der Sprache des Host-Computers angezeigt.

TASTEN

Wenn ein Anwender die Windows-Taste mit einer Funktion verbindet, dann führt bei jedem Drücken dieser Taste das Programm die gewünschte Funktion aus und wechselt zu Windows und startet das Start-Menü.

Bei der Passworteingabe sind Klein- und Großschreibung zu beachten.

2)  INSTALLATION
---------------------------------------------------------------------------
Insérez le CD 1 d'Apocalyptica(tm) dans votre lecteur de CD-ROM ou DVD-ROM. Après quelques secondes, le menu d'exécution automatique apparaît. Cliquez sur Installer pour exécuter le programme d'installation et suivez les instructions à l'écran.

Si le menu d'exécution automatique n'apparaît pas après l'insertion du CD : 
Double-cliquez sur l'icône Poste de travail, puis sur l'icône du CD/DVD-ROM et enfin sur le fichier Autorun.exe pour accéder au menu d'exécution automatique.

Vous serez d'abord invité à sélectionner le chemin et le répertoire d'installation. Le chemin d'installation par défaut est C:\Program Files\Konami\Apocalyptica. Le programme d'installation propose à l'utilisateur d'installer DirectX 8.1. Installez cette version si une version précédente de DirectX est installée sur votre ordinateur.

Notez que ce jeu ne sera pas installé si vous tentez de l'installer dans un répertoire comportant 256 caractères ou plus.

Vous pouvez commencer à jouer à Apocalyptica(tm). 

Remarque : si vous utilisez Windows XP ou Windows 2000, vous devez détenir les droits administrateur pour installer et exécuter Apocalyptica(tm).
  

3)  MENU D'EXÉCUTION AUTOMATIQUE
---------------------------------------------------------------------------
Le menu d'exécution automatique apparaît chaque fois que le CD 1 d'Apocalyptica(tm) est inséré dans le lecteur de CD/DVD ROM. Ce menu comporte plusieurs options :

*	Play (Jouer) - uniquement disponible si vous avez installé le jeu. Cliquez sur cette option pour commencer à jouer à Apocalyptica(tm). 
*	Install (Installer) - uniquement disponible si vous n'avez pas installé Apocalyptica(tm) sur votre ordinateur. Cliquez sur cette option pour installer le jeu.
*	Uninstall (Désinstaller) - uniquement disponible une fois qu'Apocalyptica(tm) a été installé sur votre ordinateur. Cliquez sur cette option pour désinstaller le jeu.
*	Readme (Lisez-moi) - instructions.


4)  DÉMARRAGE DU JEU
---------------------------------------------------------------------------
Pour commencer à jouer à Apocalyptica(tm), insérez le CD 1 dans le lecteur de CD-ROM ou DVD-ROM et sélectionnez Play (Jouer). Vous pouvez également cliquer sur l'icône Apocalyptica(tm) qui se trouve dans le dossier Konami\Apocalyptica (menu Démarrer - Programmes - Konami -  Apocalyptica). 
 

5)  DÉSINSTALLATION DU JEU
---------------------------------------------------------------------------
Si vous souhaitez désinstaller Apocalyptica(tm), vous pouvez procéder de l'une des manières suivantes :

*	Insérez le CD 1 d'Apocalyptica(tm) et sélectionnez Uninstall (Désinstaller) dans le menu d'exécution automatique.
*	Cliquez sur le bouton DÉMARRER, choisissez Programmes, Konami, Apocalyptica(tm) et cliquez sur Uninstall Apocalyptica(tm) (Désinstaller Apocalyptica(tm)). 
*	Cliquez sur le bouton DÉMARRER, allez dans Panneau de configuration, Ajouter/Supprimer et choisissez Apocalyptica(tm) pour désinstaller le jeu.
 

6)  MÉMOIRE ET PERFORMANCES
---------------------------------------------------------------------------
Apocalyptica(tm) nécessite un minimum de 128 Mo de RAM et un processeur Pentium(r) III cadencé à 800 MHz. Les performances de ce jeu seront meilleures si vous avez plus de RAM, et un processeur et une carte graphique plus rapides. Quelle que soit la configuration de votre ordinateur, vous pouvez optimiser la mémoire disponible et améliorer les performances globales du jeu en procédant comme suit : 

*	Fermez tous les programmes Windows qui ne vous servent pas, notamment les programmes de chat en ligne, les détecteurs de virus et les utilitaires d'agencement de l'écran.
*	Défragmentez régulièrement votre disque dur.
*	Faites de la place sur votre disque dur.


7)  FONCTIONS DE SÉCURITÉ DU CD D'APOCALYPTICA(tm)
---------------------------------------------------------------------------

Apocalyptica(tm) a besoin de lire des données sur le CD 1 de temps en temps. Assurez-vous que le CD 1 est toujours inséré dans votre lecteur de CD. Si vous retirez le CD, vous risquez d'activer des fonctions de sécurité qui affecteront les performances d'Apocalyptica(tm).


8)  GRAPHIQUES 
---------------------------------------------------------------------------

Apocalyptica(tm) nécessite une carte graphique à accélération 3D et d'au moins 32 Mo de RAM. Les lecteurs de votre carte graphique doivent prendre DirectX 8.1 ou supérieur en charge. Si vous rencontrez des problèmes d'exécution du jeu, procurez-vous les pilotes les plus récents pour votre carte graphique avant de contacter le service d'assistance technique.

Si vous activez la fonction Gestion de l'énergie dans le système Windows ME, le jeu n'affiche pas les graphiques correctement et finit par planter.

Les utilisateurs ne peuvent modifier la qualité des couleurs que dans les propriétés d'affichage qui se trouvent sous Paramètres - Panneau de configuration - Affichage - Propriétés d'affichage.

Si le jeu est trop sombre, désactivez " Lumières dynamiques " dans le menu des options de l'onglet Vidéo.

REMARQUE : cartes 3Dfx - les cartes 3Dfx ne sont pas officiellement prises en charge. Malheureusement, 3Dfx ne fournit plus de pilotes actualisés et ne prend plus en charge certaines fonctions de DirectX 8.
Vous pourrez peut-être trouver un pilote récent sur le site Web suivant : http://www.voodoofiles.com/.
 
REMARQUE : si vous choisissez d'installer de nouveaux pilotes 3Dfx, vous le faites à vos risques et périls !


9)  MULTIJOUEUR ET TOUCHES
---------------------------------------------------------------------------

MULTIJOUEUR

Les utilisateurs qui jouent avec un pare-feu ou un routeur ne pourront pas jouer en mode multijoueur.

Lorsque vous jouez dans une partie multijoueur, tout le texte affiché apparaît dans la langue de l'ordinateur hôte.

TOUCHES

Si un utilisateur affecte une touche Windows à une commande, chaque fois qu'il appuiera sur cette touche, le jeu exécutera la fonction souhaitée et passera dans Windows pour lancer le menu Démarrer.

Tous les mots de passe respectent la casse.
